# Jeffrey Website

A modern, responsive website for the Jeffrey Java Profiling Tool built with Vue.js and Bootstrap.

## Description

This website serves as the official landing page for Jeffrey, a powerful Java profiling tool that works with JFR (Java Flight Recorder) recordings. The site provides information about the tool, download options, and comprehensive documentation.

## Features

- Responsive design using Bootstrap 5
- Interactive UI components with Vue.js
- Markdown content rendering for documentation
- Smooth scrolling navigation
- Mobile-friendly layout

## Project Setup

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `public/introduction/`: Contains markdown files with project documentation
- `src/components/`: Vue components
- `src/assets/`: Static assets
- `src/App.vue`: Main application component
- `src/main.js`: Application entry point

## Technologies Used

- Vue.js 3
- Vite (build tool)
- Bootstrap 5
- Bootstrap Icons
- Marked (markdown parser)
