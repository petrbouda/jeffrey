Index page — ecosystem panel mockups

Five design directions for a new panel row below the Microscope + Hub dual hero, surfacing the four sub-projects: Provisioner, IntelliJ Plugin, Performance Analyst (New / Incubating) and Jeffrey JIB. Each mockup recreates the dual hero on top so you see the panels in context.

1

Glass on dark band

Glassmorphism cards on a dark gradient band that continues the hero. Per-card colored glow on hover.
2

Ecosystem light grid

Clean light section with colored top-border cards, category pills, and a "New" corner ribbon.
3

Feature strip

Tall vertical cards, big gradient icons, animated underline; Performance Analyst card elevated.
4

Bento mosaic

Bento layout — one large highlighted Performance Analyst tile plus three gradient tiles.
5

Dev-tool ribbons

Dark developer-tool panels with corner ribbons and monospace command hints (provisioner init, jib:build).

Standalone HTML previews — not wired into the app. After you pick a direction, the chosen design gets ported into jeffrey-pages/src/views/HomeView.vue using the existing scoped-CSS patterns.
